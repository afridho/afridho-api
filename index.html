afridho-api

Pushover

send notification to Pushover by QueryParam. *All the parameters must be encoded

Method: GET

endpoint: /pushover?{param}
parameters:
  • title= title notification
  • message=message
  • url = url
  • url_title = title of url
  • device = empty this(all devices)
  • priority = [low, normal, high, risk]
  • status = success or failure (error) will show icon status
Response
{
    "status":"Notification sent.",
    "title":"Pushover Api",
    "message":"This is a message",
    "device":"ridhosmac"
}
